
# 專案名稱

## Duty-Event-Recorder(值班事件紀錄工具)

一個用於中控室 / 值班人員的簡易事件紀錄工具。

## 📌 專案背景
在保全／中控值班過程中，事件通常以紙本或零散備忘方式記錄，
容易遺漏、交班困難，因此開發此工具進行本機化事件紀錄。

## 🛠 使用技術
- Python 3
- Tkinter（GUI）
- 本機檔案儲存（CSV / TXT）

## 功能
- 記錄值班事件（時間自動產生）
- 支援消防 / 電力 / 弱電 / 其他事件分類
- CSV 格式儲存，可用 Excel 開啟

## 適合對象
- 保全人員
- 中央監控室
- 機電值班工程人員

## 技術
- Python 3
- CSV 檔案存儲

## ▶ 使用方式
- duty_log_gui
